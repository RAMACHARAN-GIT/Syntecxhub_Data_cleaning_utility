## Data Cleaning Utility
## Overview

I built this project to automate common data cleaning tasks and make datasets ready for analysis or machine learning.

---

## What it does
Handles missing values (drop / fill / impute)
Fixes incorrect data types (numbers, dates)
Parses date columns
Removes duplicate records
Standardizes column names (clean + snake_case)
Generates a simple cleaning log

---
## Tech Used
Python
Pandas
NumPy

---

## Output
Cleaned dataset
Cleaning log with steps applied
